

## JS type to vim type


| JS type   | Vim type        |
|-----------|-----------------|
| string    | String          |
| number    | Number OR Float |
| boolean   | Boolean         |
| null      | v:null          |
| undefined | No equivalent   |
| Object    | Dictionary      |
| Array     | List            |
| function  | Funcref         |

ref: https://w0rp.com/blog/post/vim-script-for-the-javascripter/
